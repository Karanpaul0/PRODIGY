Project Three

This Python project is a basic implementation of the classic "Rock, Paper, Scissors" game. In this game, the user plays against the computer, with both making a choice between rock, paper, or scissors. The computer’s choice is randomly generated, while the user provides input through the keyboard. The game then compares the choices based on the rules of "Rock, Paper, Scissors" to determine the winner. The program also handles cases where the result is a draw, or one of the players wins or loses, and displays an appropriate message for each outcome.
